# Concept 06 — terminal-cyberdeck

Credit fuels machines. The brand-world is a Bloomberg Terminal that escaped to a cyberdeck enthusiast's basement. CRT phosphor green on black (or amber on black — pick one), ANSI box-drawing characters as UI primitives (┃ ━ ╋ ┏ ┓), terminal monospace (Berkeley Mono, Departure Mono, Iosevka Term), hex addresses everywhere, soldered-PCB silkscreen aesthetic.

Charts are ASCII bar graphs. Headers are box-drawn. Cursor blinks. There is a fake F1-F12 function-key bar at the bottom. The logo is a soldered-PCB pad-and-trace pattern that spells "CM" in copper.

References: the original Bloomberg terminal of 1981, the cyberdecks subreddit, Windows 3.1 dialog boxes, Final Cut Pro's old waveform UI, the C64 BASIC prompt, the "Hackers" (1995) Acid Burn UI.

Credit, in this world, is fuel for autonomous machines — and the UI is for the machines, not the humans. Humans only watch.
