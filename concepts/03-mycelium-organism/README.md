# Mycelium / Organism

## Thesis

A protocol is not a product. It is a substrate. Credit, in this world, is the
mycelium — the white-thread organism beneath the forest floor that decomposes
rent into nutrients and routes sugar between trees that have never met. You
don't see it. You see the mushrooms — the agents — fruiting briefly above
ground. You see the spores — the lenders — drifting in. The protocol is
everything between.

CredMesh, drawn this way, looks like a 19th-century field guide kept by a
mycologist who happens to also know what a Squads multisig is. The dashboard
is a herbarium sheet. A pool is a colony. An advance is a fruiting body. A
default is a fairy ring — visible only as absence. Reputation is the slow
darkening of the substrate where a hypha has passed many times.

The brand does not perform "intelligence." The forest is intelligent and
doesn't tell you. The reader is trusted to be patient, literate, and curious.
Numbers are set in serifs. Charts are hand-drawn ink. Margins are wide
because a real field guide leaves room for the reader's pencil.

## Design rules

1. **Serifs only.** Body in a transitional serif (Source Serif, Lyon Text). Display in a 19th-century display serif (Caslon, Tiempos Headline). One italic for botanical Latin. Sans-serif is forbidden — even for numbers.
2. **No straight lines except gridded margins and rules.** Charts are hand-drawn paths. A utilization curve is a hyphal tendril, not a polyline. Tabular rules are 0.5pt hairlines, never bars.
3. **No circles as primary shapes.** Round forms are spores — small, scattered, secondary. A pie chart is disqualifying.
4. **Palette is from the forest floor.** Parchment cream `#F2EBDA`, peat ink `#1B1812`, deep moss `#3F4A2A`, ochre `#B5832A`, chartreuse-lichen `#A8B560`, oxblood spore-print `#7A2A1F`. No pure white. No pure black. No accent blue, ever.
5. **Every primary number is annotated.** Like a Haeckel plate: a numeral, a tic mark, a Latin or Latin-styled epithet underneath. `$11.2M / Crescentia substrati / total deposited`.
6. **Latin binomials replace navigation labels.** Pool → *Substratum*. Agent → *Sporophorum*. Advance → *Carpophorum*. Reputation → *Patina*. Used sparingly — a glossary is allowed but the human label is always second, smaller.
7. **Diagrams are illustrations.** All visuals look hand-engraved: stippling for fill, cross-hatching for shadow, contour lines that vary in weight. Never solid fills, never gradients, never glow.
8. **Page is a herbarium sheet.** A printed gridded margin in the lightest moss, a specimen number top-right (`Tab. LXIII`), a date in roman numerals, a single hairline rule between sections. Generous whitespace — at least 96px gutters at 1440.
9. **Motion, when it appears, is biological.** Things grow and decompose. Nothing slides, fades, or "transitions." A new advance unfurls; a closed one is overgrown.
10. **The logo is a specimen, not a mark.** Drawn, not constructed. Annotation tics and a numeral are part of the lockup. It can be reproduced in pencil and still be itself.
