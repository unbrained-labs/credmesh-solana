# Concept 07 — Cartographic Topology

## Thesis

Credit has terrain. A liquidity pool is a watershed; utilization is its elevation profile; an agent's reputation is altitude above sea level; a default is a crevasse, marked with the inward-facing hachure ticks cartographers use for a depression. CredMesh is not a DeFi dashboard. It is a USGS 7.5′ quadrangle sheet, edition 2026, of a country that does not appear on any other map.

The product is for autonomous agents; the brand is for the human reading the chart before letting the agent walk into the wilderness. Pool fees scale with utilization the way air thins with altitude — quoted on the legend, not negotiated. Reputation is a benchmark elevation: stamped, dated, recoverable. The lineage is single: USGS, Imhof's Swiss Manner, Tufte. Cream paper, sepia ink, cobalt for hydrography only — the blue of water, never of a button.

Tagline: **CredMesh — A survey of the credit terrain.**

## Design rules

1. **Two ink colors plus paper.** Sepia `#7A4E2D` for line-work; cobalt `#1B3A6F` for hydrography and the single live series; paper `#EFE3CB` behind. Crevasse-red `#6B2520` is reserved for defaults.
2. **Never a circle as a primary shape.** Circles are reserved for the benchmark disc and control points. Data is contours, hachures, or labelled columns — never donuts, pies, bubbles.
3. **Labels are ALL CAPS, letter-spaced ~0.10em, condensed bold sans (Barlow Condensed / Trade Gothic).** Mixed case only inside prose blocks.
4. **Numerals are JetBrains Mono, tabular lining figures.** A number on a map is a survey reading; it aligns in columns.
5. **Every screen has marginalia.** Title block, scale bar, north arrow, declination, contour interval, datum, edition date. No margin, not a chart.
6. **Hairline rules, never boxes.** 0.5–1px sepia. No filled containers, no shadows, no rounded corners. Survey sheets fold flat.
7. **Defaults render as crevasses.** Inward-facing hachure ticks, three to seven, labelled with the survey date. Risk drawn the way a map draws a cliff.
8. **No animation except on update.** A changed value gets a hand-stamped overprint: old value struck through, new value below with a `REVISED` tag. Nothing fades.
9. **Reputation is elevation; never call it a score.** `ELEV. 1,847 m`. The agent is a control point recovered in the field.
10. **The logomark is the benchmark disc.** Never a wordmark alone, never a flat C-in-a-circle. Struck into bronze, not exported from Figma.
