# 04 — TYPOGRAPHY, CINEMATIC

## Thesis

Credit, in this world, is a **cinematic statement** — set in type so heavy and so confident it can be read from across a trading floor, undeniable. CredMesh isn't a fintech with a glyph and a gradient. It's a Saul Bass title card. A Vignelli annual report. A Müller-Brockmann poster that happens to settle on-chain.

There is no logomark. There is no icon. There is no colored circle with three letters in it. The **wordmark IS the identity** — CREDMESH set in the heaviest neo-grotesk in the room, locked to a grid, dropped onto paper-white with a single violent red mark to mean *this is the line of credit*. The protocol's product is for autonomous machines, but its dashboards have the graphic dignity of an Otto Preminger one-sheet.

Numbers are cast as the lead actor. The TVL is 180pt. The utilization rate is a single oversized digit, set in red, that takes up a quarter of the screen. The recent-advances table reads like a film's closing-credit roll: timestamp, agent, amount, *passed*, *fin*. Every screen is a frame. Every dashboard is a poster. The protocol is a film, and we are still shooting.

## Design rules

1. **No glyph. No icon. No logomark.** The wordmark IS the identity. CREDMESH is set in the heaviest neo-grotesk in the system, all caps, tracking -0.04em. No alternate marks exist.
2. **Three colors only.** Paper white `#F2EFE7`, ink black `#0A0A0A`, violent red `#E2231A`. No grays except as low-opacity ink. No gradients. Ever.
3. **One red element per frame.** Red is reserved for the single number or word that must dominate. Decorative red is forbidden — red is a knife, not a paintbrush.
4. **Numbers are the lead actor.** The most important number on any frame is set at ≥160pt with tabular numerals. Anything competing with it is downsized until it doesn't.
5. **Type sizes follow a fixed 1.5× scale:** 11 / 16 / 24 / 36 / 54 / 80 / 120 / 180 / 240. Off-scale sizes are not negotiable.
6. **Every screen is a frame.** The bottom-right of every page carries a sequential frame counter (`FRAME 01 / 47`) in 11pt small caps. The protocol is a film; we are still shooting.
7. **Tables have no borders.** Hierarchy lives in weight, leading, and tracking — never in chrome. Numerical columns right-align on tabular numerals. Ranks are set in mid-grey ink, never decorated.
8. **Headers in micro-caps, content in display weight.** Section labels are 10–11pt, all caps, tracked 0.18em. Section content is 18–240pt at weight 900. There is no middle weight.
9. **The grid is visible.** 1px ink hairlines stay in production, not stripped at handoff. No cards. No shadows. No rounded corners. The grid carries the design; nothing else may.
10. **The tagline is a block, not a sentence.** Set heavy, broken hard, never lowercased. Always: **CREDIT, COMPOSED.**
